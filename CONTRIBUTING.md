# Contributing to cyrius-doom

## Important

This is a **clean-room implementation**. Do NOT reference or copy from the id Software DOOM source code.
Use only documented specs (Unofficial DOOM Specs, DOOM Wiki, DOOM Black Book).

## Development

1. Install Cyrius compiler (cc2)
2. Obtain DOOM1.WAD (shareware — freely distributable)
3. `cyrb build` to compile
4. `./build/doom doom1.wad` to run
